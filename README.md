# PET-fMRI Metabolic Connectivity Analysis

MATLAB implementation of a graph-theoretical framework for investigating the relationship between **metabolic connectivity** derived from dynamic **[^18F]FDG PET** imaging and **functional connectivity** obtained from resting-state **fMRI**.

The project was developed as part of the *Imaging for Neuroscience* course at the **University of Padova**. The implemented pipeline reproduces the complete analysis requested in the assignment, from PET quantification using Patlak graphical analysis to the comparison of metabolic and functional brain networks through graph-theoretical measures.

---

## Project Overview

Brain connectivity can be investigated from different perspectives. While resting-state fMRI provides information about the temporal synchronization of neural activity, dynamic FDG-PET offers insight into regional glucose metabolism, a proxy for neuronal activity.

The objective of this project is to compare these complementary modalities by constructing and analysing three different connectivity networks:

- **MC<sub>Ki</sub>**: metabolic connectivity derived from regional glucose uptake rates (Ki)
- **MC<sub>TAC</sub>**: metabolic connectivity derived from PET time-activity curves (TACs)
- **FC**: functional connectivity derived from resting-state fMRI

The resulting networks are compared using graph-theoretical metrics to investigate whether metabolic and functional organization exhibit similar topological properties.

---

## Analysis Pipeline

The MATLAB script implements the complete workflow described in the assignment.

### 1. Tissue Segmentation

For every subject, the T1-weighted image registered to PET space is segmented into

- Grey Matter (GM)
- White Matter (WM)
- Cerebrospinal Fluid (CSF)

using **SPM12**.

The segmentation masks are subsequently used to exclude non-grey matter voxels during PET analysis.

---

### 2. Brain Mask Construction

The motion-corrected dynamic PET acquisition is loaded and summed across all temporal frames to obtain a static PET image.

A final brain mask is generated by combining:

- PET intensity information
- GM/WM/CSF segmentation
- exclusion of voxels with

  - WM probability > 90%
  - CSF probability > 90%
  - voxels outside the brain

Only voxels satisfying these conditions are retained for subsequent analyses.

---

### 3. Regional PET Time-Activity Curves (TACs)

Using the Schaefer atlas (100 cortical parcels), the average PET Time-Activity Curve (TAC) is extracted for every ROI.

The implementation:

- removes excluded voxels
- averages voxel-wise TACs within each parcel
- removes non-physiological values
- stores both ROI-average and voxel-wise TACs

These TACs constitute the basis for the metabolic analyses.

---

### 4. Patlak Graphical Analysis

Regional glucose uptake is quantified using **Patlak's graphical method**.

The pipeline:

- loads the provided Image Derived Input Function (IDIF)
- interpolates the IDIF to the PET temporal grid
- computes the plasma integral
- estimates the irreversible uptake rate (**Ki**) for every ROI through linear regression after selecting the linear Patlak region.

Finally, each subject's Ki values are normalized using **z-score normalization**.

---

### 5. Across-Subject Metabolic Connectivity (MCKi)

The z-scored Ki vectors from all subjects are combined to compute an across-subject metabolic connectivity matrix.

Connectivity is defined as the **Spearman correlation** between every pair of cortical regions.

The diagonal is removed since self-connections are not informative.

---

### 6. Time-Series Metabolic Connectivity (MCTAC)

An alternative metabolic connectivity measure is computed directly from PET dynamics.

For each subject:

- Euclidean distances are calculated between every pair of ROI TACs
- distances are rescaled to the [0,1] interval
- similarities are obtained as

```
Similarity = 1 − Distance
```

- Fisher's z-transform is applied

The resulting subject-specific matrices are averaged to obtain the group-level **MCTAC** matrix.

---

### 7. Functional Connectivity

The provided resting-state fMRI functional connectivity matrices are processed by:

- applying Fisher's z-transformation
- averaging the transformed matrices across subjects

to obtain a group-level functional connectivity matrix.

---

### 8. Network Thresholding

To facilitate graph analysis, each connectivity matrix is sparsified by retaining only the strongest **20%** of edge weights.

Connections below the threshold are set to zero.

The following matrices are thresholded:

- MCKi
- MCTAC
- FC

---

### 9. Graph-Theoretical Analysis

The Brain Connectivity Toolbox is used to compute two standard graph metrics:

### Node Degree

Measures the overall connectivity strength of each node.

### Weighted Clustering Coefficient

Measures the tendency of neighbouring nodes to form locally interconnected clusters.

Both metrics are computed for:

- MCKi
- MCTAC
- FC

---

### 10. Network Comparison

To investigate similarities between network organizations, node degree values are compared using Spearman correlation.

The following comparisons are performed:

- MCKi vs MCTAC
- MCKi vs FC
- MCTAC vs FC

Scatter plots with fitted regression lines are generated to visualize these relationships.

---

### 11. Hub Detection

Network hubs are defined as regions simultaneously belonging to the top **30%** of both:

- Node Degree
- Weighted Clustering Coefficient

Hub regions are identified independently for:

- MCKi
- MCTAC
- FC

allowing qualitative comparison of highly connected brain regions across metabolic and functional networks.

## Dataset

The dataset used in this project consists of dynamic **[^18F]FDG PET**, structural MRI, resting-state fMRI, atlas registrations, and auxiliary files for multiple healthy subjects.

**The dataset is not included in this repository because it is not publicly distributable.**

It can be shared **upon request**, subject to the original data-sharing policies and permissions.

The repository includes all MATLAB scripts necessary to reproduce the complete analysis once the dataset is available.

---

## Requirements

The project was developed in **MATLAB** and requires:

- MATLAB
- **SPM12** (tissue segmentation)
- **NIfTI Toolbox** for MATLAB
- **Brain Connectivity Toolbox (BCT)**
- Statistics and Machine Learning Toolbox (for correlation and distance computations)

The repository includes all auxiliary MATLAB scripts used throughout the analysis. Only external third-party toolboxes (SPM12, NIfTI Toolbox and Brain Connectivity Toolbox) need to be installed separately.

---

## Outputs

The pipeline produces:

- Regional PET Time-Activity Curves (TACs)
- Regional Ki estimates
- Z-scored Ki values
- Across-subject metabolic connectivity matrix (MCKi)
- Time-series metabolic connectivity matrix (MCTAC)
- Group-average functional connectivity matrix (FC)
- Thresholded connectivity matrices
- Node degree and weighted clustering coefficient
- Correlation analyses between network measures
- Identification of hub regions
- Summary visualizations of all connectivity matrices and graph analyses
